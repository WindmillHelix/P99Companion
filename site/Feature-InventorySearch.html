Inventory Search

The Inventory tab provides instant searches across all inventory files created with the /outputfile inventory command. New and updated inventory files are reflected immediately.

Using custom file names that do not end in -Inventory.txt will not be detected.

The Character list allows filtering to the selected character, while the Search box allows searching by item name. Putting a purely numeric value in the search box will search on the Item ID instead of the name.

When a single character is selected, the Keys button will become enabled. Clicking it will display this window detailing various key pieces and their locations.
