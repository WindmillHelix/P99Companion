Spellbook Search

The Spellbook tab provides instant searches across all spellbook files created with the /outputfile spellbook command. New and updated spellbook files are reflected immediately.

Using custom file names that do not end in -Spellbook.txt will not be detected.

After a character is selected, the spell list will populate. The search box allows searching by spell name while the Show list allows filtering to either All, Have, or Missing. Double clicking on an entry will launch the wiki page for that spell in your browser.
